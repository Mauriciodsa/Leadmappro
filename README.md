# LeadMap Pro

Aplicacao Vite + React pronta para deploy na Vercel, com funcoes serverless em `api/`.

## Rodar localmente

```bash
npm install
npm run dev
```

## Deploy na Vercel

Conecte este repositorio no painel da Vercel usando a raiz do projeto:

```txt
Framework Preset: Vite
Build Command: npm run build
Output Directory: dist
Install Command: npm install
```

Configure as variaveis de ambiente no painel da Vercel conforme o arquivo `.env.example`.
